# Logar

Simple colored logging module for python

## Features

- Default
- Success
- Error
- Warning
- Info
- Colored Logs
- Timestamped Logs

## Usage

### Install logar

```code
pip install logar
```

### Use logar

```code
from logar import Logar

log = Logar()

log.default("Default!")

log.success("Success!")

log.error("Error!")

log.warning("Warning!")

log.info("Info!")
```
